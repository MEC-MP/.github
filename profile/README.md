# Mirror's Edge Catalyst: Multiplayer

## About

Mirror's Edge Catalyst: Multiplayer (MEC: MP) is an open source community project focused on expanding what is possible with Mirror's Edge Catalyst.

[Join the discord](https://discord.gg/qBr2MAvpnt)

---

## Goals
### Multiplayer
Restore and expand online functionality for the game.
This includes:
* Player sync
* Entity replication
* World networking
* Session management
* Server infrastructure
* P2P multiplayer

### Open source SDK and modding tools
All projects relating to this project is and always will be open source and fully transparent with users.

---

## Requirements
MEC: MP requires a legitimate installation of **Mirror's Edge Catalyst** and an EA/Steam account that owns the game. The project's multiplayer systems are designed to work alongside the game's existing ownership and authentication systems. MEC: MP is not intended to replace or bypass those systems.
